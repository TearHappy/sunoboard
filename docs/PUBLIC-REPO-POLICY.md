# Public Repo Policy

This repo is public-facing documentation for SunoBoard. It exists for discoverability, support, transparency, and LLM-readable product context.

## Allowed

- README content
- Product screenshots
- Demo GIFs and videos
- Feature lists
- Public roadmap
- FAQ
- Privacy and safety explanations
- Website links
- Chrome Web Store links
- Support information
- Non-affiliation disclaimer

## Not Allowed

- Private extension source code
- API implementation details
- Secrets, keys, tokens, or private configuration
- Private build scripts
- Unreleased internal architecture
- User data
- Test accounts
- Anything copied from a private runtime environment

## Required Language

Every public surface should make these points clear:

- SunoBoard is independent.
- SunoBoard is not affiliated with, endorsed by, or sponsored by Suno Inc.
- Downloads and exports happen locally in the browser.
- SunoBoard does not store user songs on its servers.
- SunoBoard claims no ownership over user-created songs or videos.

## Feature Claim Standard

Public claims must be tied to the website, Chrome Web Store listing, privacy policy, or verified product behavior. Do not describe platform publishing features unless the current public website and product copy both support that exact claim.
